# HTML Editor - Device Gradient Separation Fix

Fix:
- Gradients/backgrounds can now be different on Mobile Edit and Desktop Edit.
- Applying a gradient on mobile changes only mobile.
- Applying a gradient on desktop changes only desktop.
- Reset/remove gradient affects only the current mode.
- Desktop reset does not remove mobile gradient.
- Mobile reset does not remove desktop gradient.
- Exported HTML includes the responsive background override.
- Keeps clear move handle, logical grouping, soft undo, carousel fixes, transparent PNG, ZIP import, and autosave.
