# HTML Editor - Canva-like Text Effects and Animations

Added:
- Floating text toolbar when a text element is selected.
- Font family, font size +/- controls, text color, bold, italic, underline, alignment in the top toolbar.
- Text transparency control.
- Letter spacing control.
- Text effects: shadow, glow, outline, background pill, neon, echo.
- Effect intensity slider.
- Text animations: keyboard, rise, slide, pop, fade, pulse.
- Export includes animation CSS.
- Keeps real device background/gradient separation, move handle, carousel, ZIP import and autosave fixes.
