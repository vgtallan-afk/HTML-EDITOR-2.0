# HTML Editor - Device Layout Separation + Basic Elements

Fixes:
- Content is shared between desktop and mobile.
- Mobile layout is snapshotted before switching to desktop, so desktop spacing/size changes do not destroy the mobile version.
- Layout properties are stored as mobile-specific overrides.
- Added simple draggable elements: Text, Image placeholder, CTA Button.
- Drag them from Add into the page or click to insert.
- Image placeholder can be replaced from the Image panel or by uploading while the placeholder is selected.
- Keeps carousel swipe lock, transparent PNG, ZIP import, autosave, desktop/mobile, and full-bleed fixes.
