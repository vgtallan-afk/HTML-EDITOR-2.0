# HTML Editor - Carousel Swipe Lock Fix

Fix:
- When the user swipes horizontally inside a carousel, vertical page scroll is temporarily blocked.
- When the user stops swiping, vertical scroll works again.
- If the gesture is vertical, the page scrolls normally.
- Exported/published pages include the same carousel swipe lock fix.
- Keeps previous carousel, transparent PNG, ZIP import, autosave, desktop/mobile, and full-bleed fixes.
