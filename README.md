# HTML Editor - Clear Move Handle Fix

Fixes:
- Selecting an element now shows a clear purple move handle icon.
- Movement starts from the move handle, not from dragging text/image directly.
- This avoids accidentally selecting text while trying to move.
- Selecting one element moves only that element.
- Selecting multiple elements with Shift/Ctrl/Cmd shows one move handle and moves all selected elements together.
- Keeps logical grouping, soft undo, carousel fixes, transparent PNG, ZIP import, and autosave.
